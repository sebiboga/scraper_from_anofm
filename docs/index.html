ANOFM Scraper - Documentatie

Acest document descrie procesul automatizat de scraping pentru companiile din ANOFM.

Sumar Proces

  1. Pas 1: Extragem companiile din Solr
  2. Pas 2: Verificam ce a fost deja scrapuit azi
  3. Pas 3: Trimitem companiile spre scrapuit
  4. Pas 4: Lucram pe batch-uri
  5. Pas 5: O noua zi = o noua sesiune

Pas 1: Extragem companiile din Solr

Extragem toate companiile care au URL-uri cu "anofm" din core-ul job din Solr:

curl -s -u solr:SolrRocks "https://solr.peviitor.ro/solr/job/select?q=url:*anofm*&rows=0&facet=true&facet.field=company&facet.limit=10000"

Salvam rezultatul in companies.json sub forma de array JSON.

Pas 2: Verificam ce a fost deja scrapuit azi

Verificam workflow runs pentru azi (UTC+3 pentru Romania):

gh api "repos/peviitor-ro/peviitor_opencode_AI_scrapers/actions/runs?per_page=100" -q '.workflow_runs[] | select(.created_at >= "2026-04-11T20:00:00Z") | .id' | wc -l

Sau citim din scraped_today.json care contine lista companiilor trimise spre scrapuit.

Pas 3: Trimitem companiile spre scrapuit

Workflow folosit: .github/workflows/opencode_scraper_to_solr.yml din-repo peviitor-ro/peviitor_opencode_AI_scrapers

gh workflow run .github/workflows/opencode_scraper_to_solr.yml -f company='NUME_COMPANIE'

Parametrul company este numele firmei de trimis.

Pas 4: Lucram pe batch-uri

Folosim scriptul scrape_remaining.js care:

  • Citeste scraped_today.json pentru a afla de unde sa continue
  • Asteapta pana cand au maxim 90 de workflow-uri in paralel
  • Trimite cate o companie la fiecare 3 secunde
  • Actualizeaza scraped_today.json dupa fiecare batch
node scrape_remaining.js

Dupa fiecare batch (~27 companii), facem commit + push:

git add scraped_today.json && git commit -m "Track X companies" && git push

Pas 5: O noua zi = o noua sesiune

In fiecare zi noua:

  1. Stergem scraped_today.json (fisier nou, azi e gol)
  2. Reextragem companiile din Solr (pot fi job-uri noi)
  3. Incepem de la index 0

Fisiere importante

FisierDescriere
companies.jsonLista tuturor companiilor extrase din Solr
scraped_today.jsonLista companiilor trimise spre scrapuit azi
scrape_remaining.jsScriptul de automatizare
